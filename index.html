and here are some graph I am interested in.

this dataset is find on https://www.kaggle.com/geomack/spotifyclassification. it is a song list on sportify 2017, and it measure the the acousticness, danceability and instrumentalness of the song. I am interested in whether is any relationship among these three factors. pairwise, i plot 3 graphs here. I use scatter plot for the graph, and for vegalite, the mark is “point”. the reason i use scatter plot is there is a lot of data point in the graph, it is the best presentation.

for the first graph, acousticness against instrumentalness, there is no obvious relationship. And so does danceability against instrumentalness. However, i would say there is a very weak linear relationship for danceability against acousticness, and the slope is negative.
